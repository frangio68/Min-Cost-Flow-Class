Detailed Description

Although using macros for activating features of the implementation is not very C++, switching off some unused features may make the code more efficient in running time or memory.

◆ DYNMC_MCF_CS2

#define DYNMC_MCF_CS2   1

Decides if the graph topology (arcs, nodes) can be changed.

If DYNMC_MCF_CS2 > 0, the methods of the public interface of class that allow to change the topology of the underlying network are actually implemented. Possible values of this macro are:

  • 0 => the topology of the graph cannot be changed;
  • 1 => the methods that change the topology of the graph are implemented.
